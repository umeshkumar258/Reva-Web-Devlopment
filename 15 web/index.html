CSS Information

CSS (Cascading Style Sheets) is used to style and layout web pages.

Types of CSS

  • Inline CSS: Applied directly within an HTML element using the style attribute.
  • Internal CSS: Defined within a <style> tag inside the <head> section.
  • External CSS: Written in a separate file with a .css extension and linked to the HTML.

Advantages of CSS

  1. CSS can control the layout of multiple web pages all at once.
  2. It allows for greater consistency in design across a website.
  3. CSS reduces the size of HTML files by separating content from design.
